GITHUB COPILOT
GitHub Copilot là một công cụ hỗ trợ lập trình do GitHub và OpenAI phát triển, sử dụng trí tuệ nhân tạo để tự động gợi ý và tạo mã. Tích hợp trực tiếp vào các trình chỉnh sửa mã như Visual Studio Code, Copilot giúp lập trình viên viết mã nhanh hơn, sửa lỗi dễ dàng và nâng cao hiệu quả làm việc.

SIGN UP, SET UP AND TRY GITHUB COPILOT IN JUST 4 STEPS

STEP 1

SIGN UP

Truy cập vào đường link: Github Copilot

Sau đó chọn Get started rồi sign up vào GitHub

STEP 2

INSTALL

Khi đã vào trang web của Copilot, hãy chọn Install Copilot cùng với code editor mong muốn (trường hợp này chúng ta sẽ chọn VS Code, nếu bạn chưa có bất kì code editor nào hãy tải xuống trước). Sau đó chọn Install Github Copilot tại trang web của VS Code

STEP 3

SET UP EXTENSIONS

Web sau đó sẽ chuyển hướng tới Extensions của VS Code.

Chỉ cần chọn Install và chờ đợi. Khi đã Install xong, bạn có thể Install thêm Github Copilot Chat nếu muốn bằng cách tìm kiếm tại Extensions.

STEP 4

TRY GITHUB COPILOT

Tạo một source code file và sau đó bấm tổ hợp phím Ctrl+I.

Nhập yêu cầu của bạn vào text field và chờ Copilot xử lý.

Khi đoạn code đã được xuất ra, nếu thấy phù hợp thì bạn có thể chọn Accept để đoạn code được autofill vào và ngược lại thì hãy chọn Discard.

Ngoài ra, bạn có thể hỏi những thắc mắc và đưa ra vấn đề gặp phải cho GitHub Copilot Chat tại Chatbox bằng tổ hợp phím Ctrl + Alt + I.
